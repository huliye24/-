# 荣景文川 · Music Site

> Official music landing page for **荣景文川 Rongjing Wenchuan** — a virtual persona & culture-tech studio from China.

## About / 关于本站

**荣景文川**是一家聚焦「虚拟人格 + AI音乐 + 符号经济」的创意科技工作室。  
这个仓库用于托管我们的对外音乐主页，用来：

- 集中展示发布和 制作中的音乐项目
- 呈现虚拟人格角色的主题歌曲与世界观
- 汇总外部平台链接

本网站为静态页面，通过 **GitHub + Cloudflare Pages** 部署。

---

## Features / 功能

- 🎧 音乐项目总览（单曲 / EP / 概念企划）
- 🧬 虚拟人格 & 世界观简介
- 🔗 外链跳转：音乐平台 / 视频平台 / 社交账号
- 🗺️ 后续可扩展：歌词页、视觉设定、时间线等

---

## Tech Stack / 技术栈

- 静态网页：HTML + CSS + JavaScript
- 代码托管：GitHub
- 部署：Cloudflare Pages
